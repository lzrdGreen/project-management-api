Project Management System (PMS) App

Purpose

The PMS app is designed to provide a user-friendly, versatile project management solution. It allows users to efficiently create, view, edit, and delete tasks and projects (CRUD operations), offering both list and board views for project visualisation. The app aims to streamline project management processes while maintaining simplicity and usability.

Key Features

  • Dual view options: list and board views for projects

  • Search functionality across projects and tasks

  • Intuitive user interface

  • Streamlined project management workflows to enhance productivity

  • Task and project CRUD operations

  • Content creation and editing according to user permissions

  • Task prioritisation and categorisation

Core Philosophy

Simplicity, usability, and added value are at the heart of the PMS app. The focus is on delivering essential features that enhance project management without overwhelming users with complexity.
